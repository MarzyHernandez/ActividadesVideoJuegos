paintT: Juego paint 

Descripción:
Este código permite dibujar formas geométricas simples utilizando la 
biblioteca Turtle. Las formas que se pueden dibujar incluyen líneas, 
cuadrados, círculos, rectángulos y triángulos.

Modificaciones realizadas:

1. Nuevo color:
   - Se agregó la opción de seleccionar el color "naranja" presionando la 
tecla 'O'.

2. Dibujar un círculo:
   - Se agregó la función "circulo(start, end)" que dibuja un círculo 
utilizando el punto de inicio y el punto final especificados.
   - El círculo se dibuja utilizando la función "circle()" de la 
biblioteca Turtle.

3. Completar el rectángulo:
   - Se completó la función "rectangle(start, end)" para dibujar un 
rectángulo correctamente utilizando los puntos de inicio y final 
especificados.

4. Completar el triángulo:
   - Se completó la función "triangle(start, end)" para dibujar un 
triángulo correctamente utilizando los puntos de inicio y final 
especificados.

Instrucciones de uso:
- Al hacer clic en la pantalla, se establece el punto de inicio o se 
dibuja la forma.
- Puedes seleccionar el color del trazo presionando las teclas 'K' 
(negro), 'W' (blanco), 'G' (verde), 'B' (azul), 'R' (rojo), 'O' (naranja), 
'P' (morado), 'Y' (amarillo), 'C' (cian), 'M' (magenta) o 'I' (rosa).
- Puedes seleccionar la forma a dibujar presionando las teclas 'l' 
(línea), 's' (cuadrado), 'c' (círculo), 'r' (rectángulo) o 't' 
(triángulo).
- Para deshacer el último trazo, presiona la tecla 'u'.
